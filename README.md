## Get Started

### 1. 克隆仓库

```shell
git clone https://github.com/HonestFox101/honestfox101.github.io.git
cd honestfox101.github.io
```

### 2. 安装依赖

```shell
pnpm i
```

### 3. 启动项目

```shell
pnpm dev
```
